# Simulation and Analysis of Normal Distribution

## Overview
This project simulates a normal distribution using Python and analyzes its statistical properties.

## Features
- Random sample generation
- Mean and variance comparison
- Histogram visualization
- Theoretical PDF overlay

## Technologies Used
- Python
- NumPy
- Matplotlib
- SciPy

## Results
The simulated data closely matches the theoretical normal distribution, validating statistical concepts.

## How to Run
1. Install dependencies:
   pip install numpy matplotlib scipy

2. Run:
   python main.py
