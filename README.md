# Simulation and Analysis of Normal Distribution

## Overview
This project simulates a normal distribution using Python and analyzes its statistical properties.

## Features
- Random sample generation
- Mean and variance comparison
- Histogram visualization
- Theoretical PDF overlay

## Technologies Used
- Python
- NumPy
- Matplotlib
- SciPy

## Results
The simulated data closely matches the theoretical normal distribution, validating statistical concepts.

## How to Run
1. Install dependencies:
   pip install numpy matplotlib scipy

2. Run:
   python main.py

# Central Limit Theorem Simulation

## Overview
This project demonstrates the Central Limit Theorem using simulation.

## Features
- Simulation of Uniform and Exponential distributions
- Sampling and computation of sample means
- Visualization using histograms

## Technologies Used
- Python
- NumPy
- Matplotlib

## Results
The distribution of sample means approaches normal distribution, validating the Central Limit Theorem.

## How to Run
1. Install dependencies:
   pip install numpy matplotlib

2. Run:
   python main.py
